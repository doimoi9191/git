Bước 1: tạo resposistory trên github.com
Bước 2: Nhân bản respo trên trang github.com xuống thư mục trong máy của bạn

git clone https://github.com/doimoi9191/git.git

Bước 3: Đẩy file mới lên respo
kiểm tra trạng thái của các file
git status

Dùng lệnh git add "tên file" hoặc git add * để thêm các file mới tạo vào thư mục nằm trên máy tính của bạn.

git add XuanND.txt
git add *

Lúc này các file vừa được thêm ở trên vẫn chưa được đẩy lên trang github.com

Dùng lệnh git commit -m"nội dung ghi chú" để thêm ghi chú.

$ git commit -m"them moi file XuanND.txt"

Dùng lệnh git push để đẩy file lên cloud của github.com

$ git push
